Mi Primera Página en GitHub :

  1. Crear una cuenta en GitHub
  2. Crear un repositorio "tunombre".github.io
  3. Utilizando Visual Studio Code crea tus archivos (uno de ellos debe ser index.html)
  4. Adjuntar los archivos a tu nuevo repositorio
  5. Ir a los ajustes, luego a Branch y cambiamos la que viene por defecto(master) por gh-pages
  6. Vamo a ajustes, luego a páginas y nos muestra un link el cual nos lleva a nuestra página web
